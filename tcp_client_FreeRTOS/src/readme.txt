【*】程序简介

-工程名称：TCP_Client_lwIP_FreeRTOS
-实验平台: 野火 GDH759IMK6-GD32开发板  

【 ！】功能简介：
FreeRTOS版本为v202411.00

本例程演示开发板发送数据到服务器

实验中使用TCP协议传输数据，电脑作为TCP Server，开发板作为TCP Client

【*】注意事项：
无

【 ！】实验操作：
电脑端使用串口调试助手，选择电脑与gd32相连的COM口，设置为115200-N-8-1，
复位开发板，即可接收gd32串口发送给电脑的调试信息。

网络连接模型如下：
	 电脑<--网线-->路由<--网线-->开发板

请将电脑上位机设置为TCP Server.在main.h文件中将目标IP地址修改为您电脑上的IP地址

修改对应的宏定义:DEST_IP_ADDR0,DEST_IP_ADDR1,DEST_IP_ADDR2,DEST_IP_ADDR3,IP_S_PORT等
